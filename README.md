# 🔐 Escape Room Challenge

## 📌 Project Overview

**Escape Room Challenge** is an interactive web-based puzzle game designed to test users’ logical thinking, problem-solving skills, and time management.  
Players must solve a series of increasingly difficult puzzles within a time limit to progress through levels and ultimately "escape".

This project combines **game design, UI/UX principles, and JavaScript logic** to create an engaging and dynamic user experience.

---

## 🎯 Objectives

- To build an interactive browser-based game using core web technologies  
- To enhance logical thinking through puzzle-solving challenges  
- To implement real-time features like timers, scoring, and animations  
- To demonstrate teamwork and version control using GitHub  

---

## 🎮 Key Features

### 🧠 Gameplay
- 20 levels (Easy → Medium → Hard → Extreme)
- Unique puzzles (logic, math, patterns, decoding)
- Increasing difficulty for better engagement

### ⏱ Timer System
- Different time limit for each level
- Auto-fail when time runs out

### 🎉 Visual Effects
- Confetti animation with happy face on correct answers
- Sad animation on wrong answers
- Full-screen transitions for immersive experience

### 🔊 Sound Effects
- Different sounds for correct and incorrect answers
- Synchronized with animations

### 💡 Hint System
- Limited hints available
- Score deduction on hint usage

### 🏆 Leaderboard
- Stores player scores using LocalStorage
- Displays ranking in descending order

### 🔁 Navigation System
- Replay level option
- Next level progression
- Home navigation

---

## ⚙️ How It Works

1. User enters their name on the home page  
2. Game starts with Level 1  
3. Player solves puzzles within time limit  
4. Correct answers:
   - Show confetti + sound
   - Move to next level  
5. Incorrect answers:
   - Show sad animation + sound  
   - Retry option  
6. Final score is stored and shown in leaderboard
